## Списки
чтобы добавить ненумерованные списки необходимо выделить звездочкой (*) Например вот так:
* элемент 1
* элемент 2
* Элемент 3
Чтобы добавить нумерованные списки, необходимо пункты просто пронумеровать. Например вот так:
1. Элемент 1
2. Второй пункт
