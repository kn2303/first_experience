# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом нужно обрамить его звездочкой (*). Например *вот так*

Чтобы выделить текст жирным нужно обрамить его двойными звездочками (**). Например **Вот ЭДАК**
